# ТЗ для генерации презентации PowerPoint
## Тема: «Разработка системы учёта ИИ-подписок в IT-компании на платформе 1С:Предприятие.Элемент»

---

## ОБЩАЯ ИНФОРМАЦИЯ О СЛАЙДАХ

**Инструмент:** Microsoft PowerPoint 365  
**Формат:** 16:9 (стандарт)  
**Количество слайдов:** 14–16  
**Переходы:** Morph (Трансформация) на КАЖДОМ слайде — объекты с одинаковым именем перетекают из слайда в слайд  
**Анимация на 1-м слайде:** декоративная — плавно появляются частицы / светящиеся точки (эффект «звёздного поля» или «нейронной сети»), зацикленная

- **Подзаголовки:** Montserrat SemiBold, 24–28pt, цвет #F0F4F8

---

## ДЕКОРАТИВНЫЕ ЭЛЕМЕНТЫ (Morph-якоря)

На каждом слайде присутствуют несколько общих объектов с фиксированными именами — именно по ним PowerPoint строит трансформацию:

- **`!!circle_accent`** — большой полупрозрачный круг (акцент), меняет позицию и размер от слайда к слайду  
- **`!!line_top`** — тонкая горизонтальная линия-разделитель вверху  
- **`!!logo_bg`** — маленький логотип/иконка 1С в правом нижнем углу

## ПОДСКАЗКИ ДЛЯ AI-ГЕНЕРАТОРА

1. Для Morph: объекты с именами `!!circle_accent`, `!!line_top`, `!!logo_bg` должны присутствовать на КАЖДОМ слайде с одинаковыми именами — тогда PowerPoint автоматически анимирует их движение.
2. Заглушки для скриншотов: вставь серые прямоугольники с текстом «[СКРИНШОТ N: описание]» — пользователь заменит их реальными снимками экрана.
3. Все слайды на тёмном фоне #0D1B2A.
4. Transition на каждом слайде: Morph, Duration: 0.8s.
