# Учёт ИИ-подписок (1С:Предприятие.Элемент)

Облачное приложение на платформе **1С:Предприятие.Элемент** для учёта расходов
IT-компании на ИИ-подписки: какие сервисы оплачиваются, кем, на чей бюджет,
сколько это стоит в месяц/год и куда уходят деньги.

## Что это и как мы работаем

- Платформа **1С:Элемент** — облачная. Приложение живёт на сервере 1С, редактируется
  в браузере. Подробности и схема работы — в [docs/platform-notes.md](docs/platform-notes.md).
- Связь «компьютер ↔ облако 1С» идёт через **Git**. Claude Code правит файлы в этом
  репозитории, пользователь синхронизирует их со средой разработки 1С.
- Рабочий цикл по шагам — в [docs/workflow.md](docs/workflow.md).
