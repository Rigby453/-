### 2. ПОСТАНОВКА ЗАДАЧИ (1–2 стр.)

**2.1 Описание предметной области**  
IT-компании используют множество ИИ-сервисов (ChatGPT, Claude, Copilot, Midjourney и др.). Подписки оформляются разными сотрудниками, нет единой картины расходов, существует риск неконтролируемых автосписаний.

**2.2 Цель разработки**  
Создать облачное приложение для учёта ИИ-подписок: кто, на что, сколько платит и когда следующее списание.

**2.3 Функциональные требования**  
- Ведение справочника ИИ-сервисов с категоризацией  
- Ведение справочника подписок с указанием ответственного, отдела, тарифа, статуса  
- Фиксация фактических оплат по каждой подписке  
- Аналитические отчёты: по месяцам, по сервисам, по отделам, прогноз, предстоящие списания  

---

### 3. ОБЗОР ПЛАТФОРМЫ 1С:ПРЕДПРИЯТИЕ.ЭЛЕМЕНТ (1,5–2 стр.)

**3.1 Назначение и особенности платформы**  
- Облачная платформа для создания корпоративных приложений  
- Описание объектов в формате YAML (справочники, документы, отчёты, формы)  
- Встроенная интеграция с Git-репозиторием  
- Автоматическая генерация интерфейса по метаданным  

**3.2 Используемые виды объектов**  

| Вид объекта | Назначение |

### 4. ПРОЕКТИРОВАНИЕ МОДЕЛИ ДАННЫХ (2–3 стр.)

**4.1 Структура подсистем**  
Приложение разбито на 4 подсистемы:
- **Справочники** — нормативные данные
- **Оплаты** — документы оплаты
- **Отчёты** — аналитические отчёты
- **Сервис** — инструменты разработчика

**4.2 Справочники**

*Сервисы:*

| Реквизит | Тип | Описание |
|----------|-----|----------|
| Наименование | Строка | Название (ChatGPT, Claude…) |
| Поставщик | Строка | OpenAI, Anthropic… |
| Категория | КатегорияСервиса | LLM / Изображения / Видео… |
| СайтURL | Строка | Ссылка на сайт |

*Подписки (центральная сущность):*

| Реквизит | Тип | Описание |
|----------|-----|----------|
| Наименование | Строка | Описание подписки |
| Сервис | Сервисы.Ссылка | Ссылка на ИИ-сервис |
| Тариф | Строка | Team / Pro / Enterprise |
| Стоимость | Число | За один цикл |
| Валюта | Валюта | RUB / USD / EUR |
| ЦиклОплаты | ЦиклОплаты | Месячная / Годовая… |
| Статус | СтатусПодписки | Активна / Триал… |
| Ответственный | Сотрудники.Ссылка | Кто отвечает |
| Отдел | Отделы.Ссылка | На чей бюджет |
| ДатаСледующегоСписания | Дата | Дата автопродления |
